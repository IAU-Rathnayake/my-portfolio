# Imesh Rathnayake - ML Engineer Portfolio

Professional portfolio website showcasing Machine Learning Engineering and Data Science expertise.

## Features
- Modern, tech-focused design with dark theme
- Responsive layout for all devices
- Interactive skill bars with animations
- Project showcase with GitHub integration
- Professional certifications display
- Contact form with validation
- Downloadable CV

## Technologies Used
- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)
- Responsive Design
- CSS Grid & Flexbox

## Sections
1. Hero Section - Introduction & quick stats
2. Skills - Technical expertise with skill bars
3. Projects - ML projects with GitHub links
4. Certifications - Professional certifications
5. Experience & Education - Timeline view
6. Contact - Contact form and social links

## Setup Instructions
1. Clone this repository
2. Update personal information in `index.html`:
   - Email, LinkedIn, GitHub URLs
   - Project descriptions and links
   - Certification details
3. Replace placeholder images with actual project screenshots
4. Update the CV download function with your actual CV
5. Deploy to GitHub Pages, Netlify, or any hosting service

## Customization
- Update colors in `:root` CSS variables
- Modify project data in the portfolio section
- Add/remove certifications as needed
- Update social media links in footer
